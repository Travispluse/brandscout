## First Impressions Happen on Social Media

Before visiting your website, potential customers often check your social media profiles. An incomplete or inconsistent profile raises doubts about your legitimacy. A polished, consistent presence builds instant credibility.

Use this checklist to set up every platform correctly.

## Pre-Launch Checklist

### Brand Name Verification
- [ ] Brand name is available as a domain
- [ ] Brand name is available on all target social platforms
- [ ] Brand name doesn't conflict with existing trademarks
- [ ] Brand name is consistent across all platforms (no variations)

Use [BrandScout](/) to verify all of the above in one search.

### Visual Assets Prepared
- [ ] Logo in square format (1:1 ratio) for profile pictures — minimum 400x400px
- [ ] Banner/header images for each platform (sizes vary)
- [ ] Brand colors defined (hex codes)
- [ ] Profile picture looks clear at small sizes (mobile thumbnails)

### Bio/Description Written
- [ ] One-line brand description (under 150 characters for Instagram)
- [ ] Longer description for platforms that allow it (LinkedIn, YouTube)
- [ ] Consistent tone and messaging across platforms
- [ ] Primary keyword included naturally

## Platform-by-Platform Setup

### Instagram
- [ ] Username claimed (@brandname)
- [ ] Profile picture uploaded (your logo)
- [ ] Bio written (150 characters max)
- [ ] Website link added
- [ ] Business account enabled (for analytics)
- [ ] Category selected
- [ ] Contact buttons configured
- [ ] One initial post published (prevents the account from looking abandoned)

### X (Twitter)
- [ ] Handle claimed (@brandname)
- [ ] Profile picture uploaded
- [ ] Header image uploaded (1500x500px)
- [ ] Bio written (160 characters max)
- [ ] Website link added
- [ ] Location set
- [ ] Pinned tweet published
- [ ] Professional account enabled (if applicable)

### TikTok
- [ ] Username claimed (@brandname)
- [ ] Profile picture uploaded
- [ ] Bio written (80 characters max)
- [ ] Website link added (requires 1,000+ followers or business account)
- [ ] Business account enabled
- [ ] One initial video published

### LinkedIn
- [ ] Company page created
- [ ] Custom URL claimed (/company/brandname)
- [ ] Logo uploaded
- [ ] Cover image uploaded (1128x191px)
- [ ] About section written (comprehensive company description)
- [ ] Industry selected
- [ ] Company size indicated
- [ ] Website linked
- [ ] First update posted

### YouTube
- [ ] Channel created
- [ ] Handle claimed (@brandname)
- [ ] Profile picture uploaded
- [ ] Banner image uploaded (2560x1440px)
- [ ] Channel description written
- [ ] Links added (website + social)
- [ ] Channel trailer uploaded (optional but recommended)

### Facebook
- [ ] Business page created
- [ ] Custom URL claimed (facebook.com/brandname)
- [ ] Profile picture uploaded
- [ ] Cover photo uploaded (820x312px)
- [ ] About section completed
- [ ] Category selected
- [ ] Contact information added
- [ ] CTA button configured

### Threads
- [ ] Account created (uses Instagram credentials)
- [ ] Bio written
- [ ] Profile picture verified (syncs from Instagram)
- [ ] Introductory post published

## Post-Launch Checklist

### Consistency Audit
- [ ] All profile pictures are identical across platforms
- [ ] All bios convey the same core message
- [ ] All usernames match (or use the same modifier)
- [ ] All accounts link back to your website
- [ ] Brand voice is consistent

### Security Setup
- [ ] Two-factor authentication enabled on every platform
- [ ] Strong, unique passwords for each account (stored in a password manager)
- [ ] Recovery email/phone configured
- [ ] Admin access properly managed (don't give everyone the password)

### Monitoring Setup
- [ ] Brand mention alerts configured
- [ ] Competitor accounts identified for monitoring
- [ ] Impersonation monitoring enabled
- [ ] Response protocol established (who responds to what)

## Ongoing Maintenance

### Monthly
- [ ] Review all profiles for accuracy
- [ ] Update seasonal or time-sensitive information
- [ ] Check for impersonator accounts
- [ ] Review analytics for each platform

### Quarterly
- [ ] Update profile pictures if branding has evolved
- [ ] Refresh bios with current messaging
- [ ] Audit follower quality (remove bots if possible)
- [ ] Evaluate which platforms are worth continued investment

## Start With the Right Name

Every item on this checklist is easier when your brand name is available everywhere from the start. Run your brand name through [BrandScout](/) before setting up any profiles — one search covers domains, social handles, and trademarks.
